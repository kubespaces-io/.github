# Security Policy

Security reports for any KubeSpaces repository:

- **Preferred:** [private vulnerability reporting on kubespaces-io/kubespaces](https://github.com/kubespaces-io/kubespaces/security/advisories/new)
- **Email:** security@kubespaces.io

Never use public issues for security reports. Full policy, scope, SLAs and
safe harbor: [kubespaces-io/kubespaces/SECURITY.md](https://github.com/kubespaces-io/kubespaces/blob/main/SECURITY.md).
